# How to run the app?

### Preparation
To use Gemini AI API, provide your API key in application.properties file located in the 'resources' directory with property name `gemini.api.key`.



### Web Application
Make sure you have Node.js and npm installed on your machine. You can download them from [Node.js official website](https://nodejs.org/).

To run the web application, navigate to the 'webapp' directory and start a local HTTP server on port 3000 using the command below.

```
cd webapp

http-server -p 3000 
```


### Backend Server
Start Spring boot application by running the main class.
